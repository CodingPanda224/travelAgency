# travelAgency


Nav bar that includes the different services
 ## - Flights (Flight search API)
 ## - Accommodation (Hotel through booking or google)
 ## - Car rental (tentative)
 ## - Currency Exchange rates 
 ## - Budgeting (Total trip cost and spending tracker)
 ## - Recommendations 

 - Initially the user will determine if the trip is a one-way or roundtrip
 - then they will need to pick an origin and destination
 - then the user is prompted to review and choose from a list of available flights 
        - Is the flight non-stop or with stops
        - What time of day is preferred to travel
 - 



 -Storing (local) Saving the budget and the choices the user has selected, time (date) of travel
